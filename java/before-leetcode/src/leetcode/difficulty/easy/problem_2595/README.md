# 2595. Number of Even and Odd Bits
양수인 정수 `n`이 주어진다.  
`even`은 짝수 인덱스 중 값 `1`을 가진 `bit`의 개수를 나타냄  
`odd`는 홀수 인덱스 중 값 `1`을 가진 `bit`의 개수를 나타냄  
`answer = [even, odd]`를 반환한다.
### 입력제한
- 1 <= `n` == 1000
### 풀이핵심
- 이진수의 각 자리 값을 비교하고 자리수가 짝수인지 홀수인지를 비교해야 한다.
- 단, 이진수이기에 자리수는 `왼->오`가 아닌 `오->왼`이다.
### 문제풀이
1. `pointer`를 생성하고 `0`으로 초기화한다.
   - `0`일 경우 `bit`는 `짝수 자리`이고 `1`일 경우 `홀수 자리`이다.
2. `n`이 `0`이 될 때까지 아래의 연산을 반복한다.
   - `bit`에 `n % 2`의 값을 저장한다.
   - `bit`가 `1`이면서 `pointer`가 `0`일 경우 `even`이 `1`증가한다.
   - `bit`가 `1`이면서 `pointer`가 `1`일 경우 `odd`가 `1`증가한다.
   - `pointer - 1`의 절대값을 `pointer`에 저장한다.
     - `bit`의 자리는 `짝 -> 홀 -> 짝 -> 홀...` 반복되기 때문
   - `n / 2`의 값을 `n`에 저장한다.
3. `(2)`의 과정이 전부 끝났다면 `even`과 `odd`를 정수형 배열에 담아 반환한다.
