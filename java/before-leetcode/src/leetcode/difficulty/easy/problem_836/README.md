# 836. Rectangle Overlap
`x, y`축에서 직사각형은 [x1, y1, x2, y2] 배열로 표기된다.  
- `[x1, y1]`은 직사각형의 왼쪽하단 좌표
- `[x2, y2]`는 직사각형의 오른쪽상단 좌표  
- 직사각형이기 때문에 아랫변과 윗변은 x축에 평행하고, 왼쪽변과 오른쪽변은 y축에 평행하다.  

두 직사각형의 교차 영역이 양수이면 두 직사각형은 겹친다.  
모서리나 가장자리만 닿을 경우 직사각형은 겹친다고 할 수 없다.  

두 직사각형 `rec1`과 `rec2`가 주어졌을 때, 겹치면 `true`를 아니라면 `false`를 반환한다.
### 입력제한
- `rec1.length, rec2.length == 4`
- -10<sup>9</sup> <= `rec1[i], rec2[i]` <= 10<sup>9</sup>
- `rec1`과 `rec2`은 `0`의 면적이 아닌 유효한 면적을 가진 직사각형을 말한다.
### 풀이핵심
- 두 직사각형이 겹치기 위한 최소한의 조건을 모두 만족하도록 하는 경우가 참이다.
- `rec1`을 제자리에 둔 상태로 `rec2`를 움직여 겹치기 위한 조건 4가지를 정했다.
  1. `rec1`의 왼쪽, 오른쪽, 상단, 하단 변을 기준으로 `rec2`를 움직인다 가정한다.
  2. `rec2`를 움직일 때 움직이는 축(방향)을 제외한 축의 값은 `rec1`과 같다 가정한다.
### 문제풀이
1. `rec1`의 `x1`보다 `rec2`의 `x2`가 커야 직사각형이 겹칠 수 있다.
2. `rec1`의 `x2`보다 `rec2`의 `x1`가 작아야 직사각형이 겹칠 수 있다.
3. `rec1`의 `y1`보다 `rec2`의 `y2`가 커야 직사각형이 겹칠 수 있다.
4. `rec1`의 `y2`보다 `rec2`의 `y1`가 작아야 직사각형이 겹칠 수 있다.
