# 342. Power of Four
정수 `n`이 주어질 때, `n`이 `4`의 거듭제곱이면 `true`를 반환한다.  
만약 아니라면 `false`를 반환한다.
### 입력제한
- -2<sup>31</sup> <= `n` <= 2<sup>31</sup> - 1
### 풀이핵심
- `n`을 `4`로 반복해서 나누었을 때 최종 `몫`이 `1`이고 모든 연산에서 나누어 떨어진다면 `n`은 `4`의 거듭제곱이라 할 수 있다.
### 문제풀이
1. 만약 `n`이 `0`이라면 `false`를 반환
2. `n`이 `1`이 될 때까지 아래의 연산을 반복한다.
   - `n`을 `4`로 나눈 나머지가 `0`이 아니라면 `false`를 반환
   - 나머지가 `0`이라면 `n`을 `4`로 나눈 몫을 `n`에 저장
3. `(2)`의 과정을 마쳤다면 `true`를 반환한다.
